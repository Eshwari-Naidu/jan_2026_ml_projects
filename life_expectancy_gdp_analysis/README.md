# Life Expectancy vs GDP Analysis

## Overview
This project analyzes the relationship between GDP and life expectancy across six countries using data from the World Health Organization and the World Bank.

## Tools Used
- Python
- pandas
- matplotlib
- seaborn

## Key Findings
- Life expectancy has increased over time across all countries
- GDP and life expectancy show a positive but non-linear relationship
- Higher GDP countries experience diminishing returns in life expectancy

## Files
- `life_expectancy_gDP.ipynb`: Main analysis notebook
- `all_data.csv`: Dataset
